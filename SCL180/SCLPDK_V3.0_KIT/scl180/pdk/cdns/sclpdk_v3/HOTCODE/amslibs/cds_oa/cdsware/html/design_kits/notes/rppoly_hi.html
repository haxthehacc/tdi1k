DESCRIPTION:

'rppoly_hi' implements the high value poly resistor. In bcd25, ca13/sbl13/sbc13, sbc18h2, and ca18 this is formed by gate poly which is covered by the hvres implant blocking layer and salicide block. Large resistors can be split up into a series connection of individual resistors ('strips') that are connected in series.

ATTRIBUTES:

  • Verify: Controls the verification of the resistor. When 'Verify' is set to 'R', the resistor value and the resistor width are compared between layout and schematic. When 'Verify' is set to 'RWS', the resistor value, the resistor width and the number of strips are compared between layout and schematic.
  • Calculated Parameter: Indicates the data entry mode of the resistor calculation. A resistor is described by the attributes 'Total Resistance', 'Width', 'Strip Length' and 'Number of Strips'. Depending on the value of this field, two of the above attributes will be editable and one will be calculated : 'Resistance' calculation is based on 'Width', 'Strip Length' and 'Number of Strips'. 'Length' calculation is based on 'Total Resistance', 'Width' and 'Number of Strips'.
  • Width: Controls the width of the resistor strip
  • Strip Length: Controls the strip length of the resistor
  • Number of strips: Allows the user to specify n resistor strips that that are connected in parallel or series. In the parallel case, the total resistance is equivalent to the resistance of an individual resistor. In the series case, the total resistance is split evenly among n individual resistors.
  • Strip Connection: When this attribute is set to 'Parallel', the 'Number of Strips' field can be used to specify a parallel resistor. When this attribute is set to 'Series', the 'Number of strips' field can be used to specify a multi-strip series resistor.
  • Total Length (Read Only): Displays the length of the combined resistor strip.
  • Total Resistance: Controls the total resistance of the resistor.
  • Bottom Dummy: Creates a dummy resistor on the bottom of the resistor.
  • Top Dummy: Creates a dummy resistor on the top of the resistor.
  • Contact on Dummy: Viewable only when the dummy resistor option has been enabled. A llows user to add contacts on dummy resistor.
  • Contact Columns: Create multiple contact columns on resistor ends.
  • Show Tap Properties: Makes left, right, top, bottom, and tap extension options viewable
  • Left Tap: Builds a tap on the left side of the resistor
  • Right Tap: Builds a tap on the right side of the resistor
  • Top Tap: Builds a tap on the of top the resistor
  • Bottom Tap: Builds a tap on the bottom of the resistor
  • Tap Extension: The tap(s) are placed the specified distance away from its default location

    • NOTES:

    LVS:

    • The following attributes are checked during LVS: Resistance, Width for the option Verify="R".
    • The following attributes are checked during LVS: Resistance, Width, Strips for the option Verify="RWS".
