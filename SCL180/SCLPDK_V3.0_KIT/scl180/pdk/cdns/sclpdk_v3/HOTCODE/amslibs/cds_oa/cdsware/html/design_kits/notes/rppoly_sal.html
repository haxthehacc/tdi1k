DESCRIPTION:

'rppoly_sal' implements a low value resistor formed by gate poly implanted with P+ implant layer (including contact regions) and using the standard silicidation process. Large resistors can be split up into a series connection of individual resistors ('strips') that are connected in series.

ATTRIBUTES:

  • Calculated Parameter: Indicates the data entry mode of the resistor calculation. A resistor is described by the attributes 'Total Resistance', 'Width', 'Strip Length' and 'Number of Strips'. Depending on the value of this field, three of the above attributes will be editable and the other one will be calculated : 'Resistance' calculation is based on 'Width', 'Strip Length' and 'Number of Strips'. 'Length' calculation is based on 'Total Resistance', 'Width' and 'Number of Strips'.
  • Width: Controls the width of the resistor strips.
  • Strip Length: Controls the strip length of the resistor.
  • Number of strips: Allows the user to specify n resistor strips that that are connected in parallel or series. In the parallel case, the total resistance is equivalent to the resistance of an individual resistor. In the series case, the total resistance is split evenly among n individual resistors.
  • Strip Connection: When this attribute is set to 'Parallel', the 'Number of Strips' field can be used to specify a parallel resistor. When this attribute is set to 'Series', the 'Number of strips' field can be used to specify a multi-strip series resistor.
  • Total Length (Read Only): Displays the length of the combined resistor strip.
  • Total Resistance: Controls the total resistance of the resistor.
  • Bottom Dummy: Creates a dummy resistor on the bottom of the current resistor.
  • Top Dummy: Creates a dummy resistor on the top of the current resistor.
  • Contact on Dummy: Viewable only when the dummy resistor option has been enabled. Allows user to add contacts on dummy resistor.
  • Contact Columns: Create multiple contact columns on resistor ends.
