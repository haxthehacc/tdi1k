DESCRIPTION :

'p3p3fet_ld' implements the high voltage pMOS transistor. The gate width can be modified and wide transistors can be generated as an array of parallel fets using the 'Number of Fingers' attributes. Multiple fets connected in parallel on the same active area are called 'fingers'.

IMPLEMENTATION :

In schematic entry, be sure to have the source and bulk connected together on the symbol. This is already the case in the layout cell.

ATTRIBUTES:

  • Count: Controls the number of fet instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n fet instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Width (finger): Controls the width of a gate finger.
  • Length: Controls the gate length.
  • Number of Fingers: Controls the number of gate fingers connected in parallel in the same active area.

NOTES:

  • The total width of a transistor is given by : width = COUNT * NF * WF
  • The total number of gates of a transistor is given by : gates = COUNT * NF

LVS:

  • The following attributes are checked during LVS : width, length, number of fingers.
