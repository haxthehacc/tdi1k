DESCRIPTION:

'pnpd' implements a vertical pnp bipolar transistor. The layout is fixed with emmitter dimensions of 3u x 3u

ATTRIBUTES:

  • Multiplier: Controls the number of tranisitor instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n transistor instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
