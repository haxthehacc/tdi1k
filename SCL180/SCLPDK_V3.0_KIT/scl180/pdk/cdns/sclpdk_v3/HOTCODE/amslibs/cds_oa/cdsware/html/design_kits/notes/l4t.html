DESCRIPTION :

'l4t' implements the 4-terminal rectangular spiral inductor fabricated in thick metal3.

ATTRIBUTES :

  • Change input parameters: This button brings up the inductor calculation window.
  • Inductor Type (Read Only): Displays the type of inductor.
  • Num of Turns: Controls the total number of turns in the inductor. A quarter turn is .25, a full turn is 1. For instance, to create an inductor with four and a half turns, specify Num of Turns = 4.5.
  • Spacing: Controls the space between inductor segments.
  • Width: Controls the width of the inductor segments.
  • Xsize: The total width of the inductor in the X direction.
  • Ysize: The total height of the inductor in the Y direction.
  • Toggle Circuit Params Display:

NOTES:

  • The inductor pcell generates a marking layer around the layout. This marking layer is used to identify the inductor during LVS.
  • It is not allowed to run any layer over or under the inductor.

LVS:

  • For LVS to recognize the inductor, the marking layer must be intersected by one metal2 wire that connects to the P1 pin and one metal3 wire that connects to P2 pin. It is not allowed to connect the pins in any other layer.
  • The following attributes are checked during LVS:

    • Width, Spacing, Number of Turns, XSize and YSize
    for sbc35mx, sbc35qx, sbc35px, bc35, bc35qx, bc35px technology runsets
