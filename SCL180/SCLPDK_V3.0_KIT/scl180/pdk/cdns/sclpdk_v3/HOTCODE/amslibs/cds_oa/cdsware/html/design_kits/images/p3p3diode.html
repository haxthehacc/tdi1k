DESCRIPTION :

'p3p3diode' implements the P+/Nwell thick oxide 3.3V junction diode.

ATTRIBUTES:

  • Style : Controls whether the interconnect scheme in the layout pcell is performed using metal 1 or metal 2 layer.
  • Specify : Controls how user wishes to specify device dimensions. A/P means specify area and perimeter, WL means specify width and length, A/R means specify desired width/length aspect ratio.
  • Bulk : Controls the presence/absence and configuration of substrate ties in the layout pcell.
  • Count : Controls the number of diode instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n diode instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic

NOTES:

LVS:

  • The following attributes are checked during LVS : area, perimeter.
