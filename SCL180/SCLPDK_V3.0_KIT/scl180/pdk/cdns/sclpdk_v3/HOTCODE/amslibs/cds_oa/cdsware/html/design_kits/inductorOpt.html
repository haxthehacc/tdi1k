Inductor Optimization

The Inductor Optimization form is a user-interface built around the Matlab Optimization algorithms for spiral and differential inductors. For details on the Matlab implementation of the optimization algorithms, please refer to the following PDF document.

The following are the available spiral inductor (Ind) and differential inductor (Diff) optimizations available for use:
  • maxIndArea: Given a specified Maximum Tolerable Capacitance per Port, compute the maximum Area that the inductor can occupy before exceeding this capacitance limit. Note that this is not an optimization algorithm but a simple analytical formula, and is useful to run before proceeding with an optimization to determine a good initial size for the inductor.
  • optIndNt & optDiffNt: Given an initial inductor area, optimize for the Quality factor of the inductor by varying Number of Turns and Width. The purpose of this optimization is to find the best inductor that can fit within a given specified area.
  • optIndQ & optDiffQ: Optimize for the Quality factor of the inductor by varying Number of Turns, Width and Area. The purpose of this optimization is to maximize the Quality Factor of an inductor.
  • optIndA & optDiffA: Given a specified target Quality Factor and Inductance, optimize for minimum Area of the inductor. The purpose of this optimization is to find the smallest inductor size for a specific Q and Inductance.
A few notes about running the optimizations from RDS:
  • Once you are happy with the results of an optimization, you must click on the OK button of the "Edit Inductor Parameters" form and then click on the OK button of the "Edit Object Properties" form in order to return the optimized parameters back to the inductor in your schematic.
  • Select the relevant algorithm (optimization) you want to run in the form and then select Run Optimization button to run an optimization.
  • In all cases, a call to the Matlab optimization executable is made and Matlab returns geometry information of the inductor. This geometry information is reported in a popup dialog when Matlab completes and is also returned to the "Edit Inductor Parameters" form as well as the "RDS Inductor Optimization" form.
  • The optimization process runs in background mode, which means you can proceed to do other things in DFII while the optimization is running. However, do not close any of the forms while the optimization is running since otherwise the geometry in the forms cannot be properly updated once the run completes.
  • If for any reason you wish to terminate the optimization that is currently running, simply click on the Kill Optimization button.
  • The OK and Cancel buttons serve no other purpose than to close the "RDS Inductor Optimization" form.
  • For all algorithms except maxIndArea, a view window showing the progress of an optimization will appear. If for some reason the optimization algorithm runs into trouble, another view window showing the error from Matlab will appear and a dialog box pops up indicating there was trouble.
