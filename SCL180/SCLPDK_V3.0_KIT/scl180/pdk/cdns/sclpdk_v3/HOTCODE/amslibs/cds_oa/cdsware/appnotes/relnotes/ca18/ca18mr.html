Relevant Specification Documents:
Design Rules (CA18*): NPB-PS-0176, rev.16
Electrical Spec. (CA18*): NPB-PS-0173, rev.14
Design Application Note (CA18* 5V): NPB-PS-0711, rev.09
Design Application Note (CA18MR): NPB-PS-1112, rev.01

Process Synopsis:
* This is an initial development-level kit release for the ca18mr
  process variant.  Issues may arise which will be addressed in
  future kit releases.
* The ca18mr process has 3 layers of aluminum metal and supports:
  - 6V FETs
  - 6V gate oxide, drain-extended 12V FETs
  - 6V low VT native NFET
  - unsalicided poly, salicided poly, and N-well resistors
  - 1fF/um2 MIM capacitor between metal2 and metal3
  - high voltage vertical NPN
  - vertical PNP
  - triple well isolation using deep N-well
  - inductors using 2.8um thick top metal3

Limitations
* Using the integrated tap option on NFETs causes DRC violations
