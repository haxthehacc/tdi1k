DESCRIPTION :

'n3p3fetd' implements the thick oxide 3.3V nMOS transistor. The gate width and length can be modified.

ATTRIBUTES:

  • Style: Specifies the layout style to be used for the device. The 'unconnected' layout style leaves the sources, drains and gates of the transistors unconnected. The '1met' style uses metal1 to connect the source and drain areas. The '2met' is the same as the 1met style for fets with 1 slice (as are all fets entered with the n3p3fetd parameter sheet.)
  • Width (finger): Controls the width of a gate finger.
  • Length: Controls the gate length.

NOTES:

LVS:

  • The following attributes are checked during LVS: width, length.
