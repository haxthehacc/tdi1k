DESCRIPTION:

'moscap' implements the thin oxide MOS capacitor (poly cap). The device is implemented as an nfet in a nwell. The top plate of the capacitor is formed by the gate, the bottom plate by the n+diffusion. The gate oxide is used as dielectric for the capacitor. The layout of the device is similar to a MOSFET with the source and drain shorted. The capacitance value can be modified by changing the gate width and gate length.

ATTRIBUTES:

  • Substrate Node: The node to which the substrate of the moscap is connected. This attribute provides an implicit substrate pin for the device.
  • Count: Controls the number of moscap instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n moscap instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Length (x): Controls the size of the capacitor fingers in the X direction.
  • Width (y): Controls the size of the capacitor fingers in the Y direction.

NOTES:

LVS:

  • The following attributes are checked during LVS: length and width.
