DESCRIPTION:

'p3p3fet_esd' implements a thick oxide 3.3V pMOS transistor with increased ESD protection. Both the nMOS and pMOS ESD fets are layed out according to the ESD rules described in the design rule document. The ESD fets should be used for all input or output signals that are directly connected to a pad.

The gate width and length can be modified, and wide transistors can be generated as an array of parallel fets using the 'Number of Fingers' and 'Number of Slices' attributes. Multiple fets connected in parallel on the same active area are called 'fingers'. Very wide transistors can be split up in multiple multi-finger sub-transistors that are connected in parallel. The resulting sub-transistors are called 'slices'. Only one layout style is available. ESD fets are surrounded by a double guard ring.

ATTRIBUTES:

  • Width (finger) : Controls the width of a gate finger.
  • Length : Controls the gate length.
  • Number of Fingers : Controls the number of gate fingers connected in parallel in the same active area.
  • Number of Slices : Wide transistors can be split up into multiple sub-transistors ('slices') connected in parallel in the same pcell. 'Number of Slices' controls the number of parallel sub-transitors.
  • Current : Specifies the current through the device. The width of the source/drain metal wires will be adjusted to carry this current.
  • Style : Controls the manner in which layout pcell interconnections are made.
  • Bulk : Controls the presence/absence and location of substrate ties in the layout pcell.

NOTES:

LVS:

  • The following attributes are checked during LVS: width(fet), length(fet), length(resistor) and width(resistor).
