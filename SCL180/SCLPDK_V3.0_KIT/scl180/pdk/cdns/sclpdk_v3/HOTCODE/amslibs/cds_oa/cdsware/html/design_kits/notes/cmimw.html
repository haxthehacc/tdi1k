DESCRIPTION:

'cmimw' implements the 1fF MIM capacitor over well.

ATTRIBUTES:

  • ToUse: Controls the way multiple parallel capacitors are specified. When 'ToUse' is set to 'Multiplier', the Multiplier attribute will be editable and the 'Rows' and 'Columns' attributes will disappear. When 'ToUse' is set to 'Row/Col' the opposite will happen. See below for the meaning of these attributes.

    • Multiplier: Indicates the number of capacitors connected in parallel. Specifying a 'Multiplier' of n is equivalent to drawing n capacitor instances and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute is available when 'ToUse' is set to 'Multiplier' and can only be used in the schematic.

    Rows: When 'ToUse' is set to 'Row/Col' the capacitor will be implemented as an an array of unit capacitors. 'Rows' specifies the number of rows of unit capacitors in the array.

    Columns: When 'ToUse' is set to 'Row/Col' the capacitor will be implemented as an array of unit capacitors. 'Columns' specifies the number of columns of unit capacitors in the array.
  • Calculate: Indicates the data entry mode of the capacitor calculation. A Capacitor is described by the attributes 'Capacitance', 'Width', 'Length' and 'Aspect Ratio'. Depending on the value of the 'Calculate' attribute, two of these attributes will be editable and the other two will be calculated : 'C' calculates 'Capacitance' and 'Aspect Ratio' based on 'Width' and 'Length'. 'L' calculates 'Length' and 'Aspect Ratio' based on 'Capacitance' and 'Width'. 'W' calculates 'Width' and 'Aspect Ratio' based on 'Capacitance' and 'Length'. 'WL' calculates 'Width' and 'Length' based on 'Capacitance' and 'Aspect Ratio'.
  • Capacitance: Controls the total capacitance value of the capacitor array.
  • Width (X): Controls the width of a unit capacitor in the capacitor array.
  • Length (Y): Controls the length of a unit capacitor in the capacitor array.
  • Aspect Ratio = w/l: Controls the aspect ratio of a unit capacitor in the capacitor array.
  • Show Contact Properties: Controls the visibility of the Contact choices for Left, Right, Top, and Bottom.
  • Left Contact: Builds contacts on the left side of the capacitor
  • Right Contact: Builds contacts on the right side of the capacitor
  • Top Contact: Builds contacts on the top of the capacitor
  • Bottom Contact: Builds contacts on the bottom of the capacitor
  • Show Tap Properties: Controls the visibility of the Tap choices for Left, Right, Top, and Bottom.
  • Left Tap: Builds tap on the left side of the capacitor
  • Right Tap: Builds tap on the right side of the capacitor
  • Top Tap: Builds tap on the top of the capacitor
  • Bottom Tap: Builds tap on the bottom of the capacitor
