(SBC18PTH) ________________ Date: 10/06/2014 Library: sbc18 Relevant Specification Documents: * Design Rules: NPB-PS-0179, Rev. 15 * Design Rules: NPB-PS-0635, Rev. 6 (SBC18PTH) * Design Rules: NPB-PS-0888, Rev. 0 (SBC18PTD) * Electrical Spec: NPB-PS-0267, Rev. 17 Process Synopsis: * The SBC18PT process has 5 layers of aluminum metal and supports: - 1.8V and 3.3V MOSFETs, - standard, high speed (SBC18PTH/PTD) and high voltage NPNs without deep trench, - poly resistors (silicided and low value and high value unsilicided), N-well resistor, - 2.0fF/um2 MIM capacitors between metal2/metal3, 4.0fF/um2 MIM stacked capacitors between metal2/metal3/metal4, - square and octagonal inductors using 5.2um thick top metal5, - MOS and high performance junction varactors (SBC18PTH/PTD) Software versions tested with this release: * Cadence IC release 6.1.6.080 * Cadence Spectre release 7.2.0.202.isr5 * Synopsys HSPICE release E-2010.12-ENG2 * Mentor Calibre release v2014.1_36 * Cadence Assura release v4.14.116_616 This is the initial release of the next-generation SBC18PT kit intended to replace the legacy kit last released in 2006.
