DESCRIPTION:

  • Contacts (PTAP, NTAP, cont) can be used for single or an array of contacts. They have the wider contact to contact spacing in a contact array larger than a 3x3.

  • The Contacts that end in "n" have a smaller contact to contact spacing rule. Use this if you need a 3x3 contact array or smaller to use the minimum space.

  • The Vias that end in "_g" have different spacing rules as well. These vias follow the ABLB rules and are array capable.

  • Multi Part Paths will be supported in SBC18 PDKs as they are released in 2007.
       A path that consists of more than one part is called a multipart path.
       The MPP command can be found under the "Create" pulldown in Layout Window.

       MPP names end with "_mpp" and can be used to create PTAP and NTAP guard rings.
          If an incomplete ring is needed, be sure to edit MPP subparts and extend implant layer to minimum enclosure or use path instead.

       MPPs can be used to create cont, v12, v23, v34 and v45 line contacts as well as a ring.
       At this time, you must bring up the MPP form to set the contact or via you desire. (double click middle mouse button to bring up menu)

       If you have access to "Layout Turbo" from your Tools pulldown menu, you can use Create->Guard Ring to easily ring your circuitry.

       All MPP vias and contacts are be choppable to allow routing if needed.
