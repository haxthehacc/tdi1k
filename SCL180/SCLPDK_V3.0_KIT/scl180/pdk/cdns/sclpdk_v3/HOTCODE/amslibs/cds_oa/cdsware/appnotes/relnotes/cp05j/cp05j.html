To set this variable, modify your project-specific .cshrc file and change the setting of RDS_CDS_VERIFY_TECH to the process variant you are using. Here are step-by-step instructions: - Go to your project workarea as usual using the source and cdsprj commands. - Type the command "cd ../../../cds_master" - Edit the file cdsPrj.cshrc that is in this directory - Locate the line containing the words "setenv RDS_CDS_VERIFY_TECH" - Change the line to the actual process variant you are using - Save your changes - Close your terminal. You must open a new terminal and start again for the changes to take effect. - source your RDS.cshrc and type the cdsprj command again. * Artifact Generator. The Artifact Generator may generate shapes that are not DRC clean. If this happens, you can either increase the scale factor or hand-edit the artifact to clean up the errors. Off-grid errors may still occur. Use Snap-To-Grid to correct these violations before hand-editing the artifact to clean it up.
