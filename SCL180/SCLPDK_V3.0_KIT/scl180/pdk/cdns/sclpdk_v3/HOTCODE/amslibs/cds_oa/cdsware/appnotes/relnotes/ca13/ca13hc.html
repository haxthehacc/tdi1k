___________________________________ Date: 11/22/06 Library: ca13 (ca13hc) Relevant Specification Documents: * Design Rules: NPB-PS-0663, rev.4 * Electrical Spec: NPB-PS-0577, rev.5 Highlights: * This is a development-level kit release for the ca13hc process variant. Issues may arise which will be addressed in future kit releases. * Added high threshold fets, metal5/6 mimcaps, square inductors, a high value resistor, and a vertical pnp * DRC and LVS updates * If you are using a ca13ha design kit, you will need to transition to this kit to see the latest changes. This kit contains the updates mentioned above and all the devices from ca13ha. * Be sure to run "Trigger Callbacks" before working with this design kit. This will allow you to see the latest changes. Software versions tested with this release: * Cadence IC releases IC5.1.41 USR3 with IUS5.7 QSR2 * Cadence AMS Designer requires use of IUS5.7 QSR2 * Cadence Spectre release MMSIM6.0 USR2 (minimum) * Synopsys Hspice release 2005.9 (minimum) * Cadence Assura 3.1.5 USR1 (minimum) * Mentor Calibre 2006.1_19.19 (minimum) Process Synopsis: * This ca13 design kit library supports only the ca13hc process variant. * The ca13hc process has 6 layers of aluminum metal. It supports the following: - 1.2V and 3.3V fets - high threshold 1.2V fets - low/high value unsalicided and nwell resistors - 3fF/um2 mim capacitors between metal 4/5 and metal 5/6 - 6fF/um2 mim capacitors between metal 4, metal 5, and metal 6 - square and octagonal inductors using 3um thick top metal 6 - 3.3V mos varactor - vertical pnp - triple well (deep nwell) * This design kit is in the development phase, so issues may arise which will be addressed in future releases. Process Variant Support: * You MUST set the environment variable RDS_CDS_VERIFY_TECH in order to use this design kit. If this variable is not set, the kit will refuse to function.
