DESCRIPTION :

'dn' implements the N+/Psub junction diode.

ATTRIBUTES:

  • Show Tap Properties: Controls the visibility of the Tap choices for Left, Right, Top, and Bottom.
  • Left Tap: Builds a tap on the left side of the diode
  • Right Tap: Builds a tap on the right side of the diode
  • Top Tap: Builds a tap on the of top the diode
  • Bottom Tap: Builds a tap on the bottom of the diode
  • Tap Extension: Increases the offset of the taps from the diode body.
  • Calculate Parameter: Controls which parameter the user wishes to calculate (area, width, length). The parameter to be calculated is not editable and the remaining two can be modified.
  • Device Area: Area of the diode
  • Width: Width of diode
  • Length: Length of diode
  • Multiplier: Controls the number of diode instances connected in parallel. Specifying a 'Multiplier' of n is equivalent to drawing n diode instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Periphery of Junction Perimeter of diode

NOTES:

LVS:

  • The following attributes are checked during LVS : area, perimeter.
