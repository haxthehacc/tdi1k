DESCRIPTION :

'dschottky' implements a schottky diode.

ATTRIBUTES:

  • Width: Width of anode active
  • Length: Length of anode active
  • Multiplier: Controls the number of diode instances connected in parallel. Specifyi ng a 'Multiplier' of n is equivalent to drawing n diode instances in the schematic and connec ting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cell s. This attribute can only be used in the schematic

    • NOTES:

    LVS:

    • The following attributes are checked during LVS : width, length.
