DESCRIPTION:

'pfet_dg' implements the pMOS dual gate (stacked) transistor configuration, which consists of two thin oxide cascoded pfets. The layout is structured such that the drain of the second (top) device "s2" overlaps the source of the first (bottom) device "d1" to minimize source diffusion area, and hence reduce parasitics on that node.

ATTRIBUTES:

  • Count: Controls the number of device instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Style: Controls if layout pcell interconnections are made, and if so, in what style of metal.
  • Width (finger): Controls the width of a gate finger.
  • Length1: Controls the length of the first fet.
  • Length2: Controls the length of the second fet.
  • Number of Fingers: Controls the number of gate fingers connected in parallel in the same active area.
  • Number of Slices: Wide transistors can be split up into multiple sub-transistors ('slices') connected in parallel in the same pcell. 'Number of Slices' controls the number of parallel sub-transitors.
  • Bulk: Controls the presence/absence and location of substrate ties in the layout pcell.
  • Current: Specifies the current through the device. The width of the source/drain metal wires will be adjusted to carry this current.

NOTES:

LVS:

  • The following attributes are checked during LVS: wf (finger width), l1 (Length1), l2 (Length2) and nf (number of fingers).
