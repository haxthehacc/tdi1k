DESCRIPTION:

'npn' implements the vertical NPN bipolar transistor. The emitter width, emitter length and number of emitter fingers can be controlled. Low and high breakdown voltage devices are available.

ATTRIBUTES:

  • Substrate Node: The node to which the substrate of the fet is connected. This attribute provides an implicit substrate pin for the device.
  • Model Type: Controls the model used to simulate the device. 'gp' selects a Gummel-Poon model, 'hicum' selects a hicum model.
  • Count: Controls the number of transistor instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n fet instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Voltage Type: Controls the npn voltage type.
  • Emitter Width: Controls the emitter finger width.
  • Emitter Length: Controls the emitter finger length.
  • Base Fingers: Controls the number of base fingers. When the number of base fingers is larger than 1, the number of emitter fingers is equal to the number of base fingers - 1. When the number of base fingers is 1, the number of emitter fingers is 1. Devices with more than two base fingers will have two collector fingers, one at the top and one at the bottom. The single emitter devices have one collector finger at the top.
  • Emitter Metal: Controls the type of metal that is used to connect the emitter and collector fingers. M1 uses only metal1. M2 uses a stack of metal1/metal2 for emitter and collector and M3 uses metal1/metal2 for emitter and metal1/metal2/metal3 for the collector. "M1 UC" is the M1 configuration with the Base and Emitter fingers left unconnected.

NOTES:

  • Not all combinations of 'Voltage Type', 'Emitter Width', 'Emitter Length' and 'Base Fingers' are available. An error message will be reported when an illegal combination is specified.

LVS:

  • The following attributes are checked during LVS: Emitter Width, Emitter Length, Base Fingers, Collector fingers, Device voltage type
