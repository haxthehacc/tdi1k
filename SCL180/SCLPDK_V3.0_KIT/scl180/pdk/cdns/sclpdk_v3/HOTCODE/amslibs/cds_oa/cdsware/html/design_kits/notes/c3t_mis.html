DESCRIPTION:

'c3t_mis' implements a capacitor fabricated using N implanted active as the bottom plate and poly as the top plate. The top plate of the capacitor can be contacted using metal 1.

ATTRIBUTES:

  • Count: Indicates the number of capacitors connected in parallel. Specifying a 'Count' of n is equivalent to drawing n capacitor instances and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute is available when 'ToUse' is set to 'multiplier'. This attribute can only be used in the schematic.
  • Bottom Contacts: Controls the location of bottom plate contacts around each unit capacitor in the capacitor array. This location is specified with a string of characters indicating the sides where bottom plate contacts have to be added: 'l' means left, 'r' means right, and 't' means top. For c3t_mis, the default value is 't' and the other possible value is 'lrt'.
  • Bulk: The capacitor pcell support the use of a shield around the array. The shield is a stack of active, contact, metal1, via1, metal2, via2 and metal3. The 'Bulk' attribute controls the location of the shield around the capacitor array. This location is specified with a string of characters indicating the sides where shields have to be added: 'l' means left, 'r' means right, 't' means top and 'b' means bottom. For instance, Bulk=l will add a shield on the left side of the array, Bulk=lbrt will add a full shield ring around the array. Bulk=n generates a device without shield.
