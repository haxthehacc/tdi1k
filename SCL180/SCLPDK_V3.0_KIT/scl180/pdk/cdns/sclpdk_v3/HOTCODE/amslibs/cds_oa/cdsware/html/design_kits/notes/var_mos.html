DESCRIPTION:

'var_mos' implements a varactor built with N+ poly and thin 1.8V gate oxide over nwell. The gate width and length control the capacitance tuning range and quality factor (Q).

ATTRIBUTES:

  • Count: Controls the number of instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Width: Controls the width of a gate finger.
  • Length: Controls the length of the gate.
  • Number of Slices: Controls the number of slices or segments connected in parallel.
  • Fingers: Controls the number of gate fingers connected in parallel in the same active area.
  • Style: Specifies the layout style to be used for the device. The 2MET_connected style uses metal2 to connect the sources/drains of all segments and the gates of all segments. The '2MET' style uses metal2 to connect the sources/drains of each individual segment and the gates of each individual segment (interconnect between segments is not drawn). Lastly the '1MET' style uses metal1 to connect the sources/drains of all segments and the gates of all segments.
  • Min/Max C:Upper and Lower bound of Cgb. Calculations are based on internal model values and user defined width and length.

NOTES:

LVS:

  • The following attributes are checked during LVS: gate width, gate length, number of fingers, number of slices, and layout style.
