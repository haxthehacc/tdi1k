DESCRIPTION:

'cpoly5p0' implements a thick oxide 5.0V MOS capacitor. The device is implemented as a thick oxide nfet in nwell. The top plate of the capacitor is formed by the gate, the bottom plate by the n+diffusion. The gate oxide is used as dielectric for the capacitor. The layout of the device is similar to a MOSFET with the source and drain shorted. The capacitance value can be modified by changing the gate width and gate length.

ATTRIBUTES:

  • Multiplier: Controls the number of cpoly5p0 instances connected in parallel. Specifying a 'Multiplier' of n is equivalent to drawing n cpoly5p0 instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Length (x): Controls the size of the capacitor fingers in the X direction.
  • Width (y): Controls the size of the capacitor fingers in the Y direction.
  • Gate Connection Style: Specifies the desired gate connection: 'Both', 'Top', 'Bottom', or 'None'.
  • Connect Source/Drain: Option to have the source and drain connected.
  • Use ABLB overplots: Controls whether ABLB overplots are used. This option is not available in all processes.

NOTES:

LVS:

  • The following attributes are checked during LVS: length and width.
