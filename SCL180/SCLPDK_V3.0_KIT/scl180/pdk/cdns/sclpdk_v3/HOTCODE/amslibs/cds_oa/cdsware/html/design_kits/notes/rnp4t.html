DESCRIPTION:

'rnp4t' implements the Low value poly resistor formed by gate poly covered with salicide block and resistor block. The explicit substrate terminal differentiates this device from 'rnp3t' device which has substrate connection as a property. The resistor is defined by the resistor block layer. Large resistors can be split up into a series connection of individual resistors ('strips') that are connected in series.

ATTRIBUTES:

  • Verify: Controls the verification of the resistor. When 'Verify' is set to 'R', the resistor value and the resistor width are compared between layout and schematic. When 'Verify' is set to 'RWS', the resistor value, the resistor width and the number of strips are compared between layout and schematic.
  • Parallel/Series: When this attribute is set to 'Parallel', the 'Count' attribute is editable and can be used to specify a parallel resistor. When this attribute is set to 'Series', the 'Number of strips' attribute is editable and can be used to specify a multi-strip series resistor.
  • On Grid: Selecting this button puts the resistor terminals on a .5um grid.
  • Count: Indicates the number of resistors connected in parallel. Specifying a 'Count' of n is equivalent to drawing n capacitor instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute is available when 'Parallel/Series' is set to 'Parallel'. This attribute can only be used in the schematic.
  • Number of strips: Controls the number of resistor strips that are connected in the series to form the resistor. If you specify number of strips equal to n, the total resistance is split evenly among n individual resistors that are connected in series.
  • Resistor Calculation: This button brings up the resistor calculation window.
  • Calculate: Indicates the data entry mode of the resistor calculator. A resistor is described by the attributes 'Resistance', 'Finger Width', 'Length' and 'Strips'. Depending on the value of the 'Calculate' attribute, three of these attributes will be editable and the other one will be calculated : 'R' calculates 'Resistance' based on 'Finger Width', 'Length' and 'Strips'. 'L' calculates 'Length' based on 'Resistance', 'Finger Width' and 'Strips'. 'W' calculates 'Width' based on 'Resistance', 'Strips' and 'Length'.
  • Finger Width: Controls the width of the resistor strips.
  • Length: Controls the total length of the resistor.
  • Resistance: Controls the total resistance of the resistor.
  • Bulk: Controls the location of substrate ties. The location is specified with a string of characters indicating the sides where substrate ties have to be added: 'l' means left, 'r' means right, 't' means top and 'b' means bottom. For instance, Bulk=l will add a substrate tie on the left side of the device, Bulk=lbrt will add a full substrate tie ring around the device. Bulk=n generates a device without substrate ties.
  • Strip Resistance (Read Only): Displays the resistance of one resistor strip.
  • Finger Length (Read Only): Displays the length of one resistor strip.

NOTES:

LVS:

  • The following attributes are checked during LVS: Resistance, Width, Length for the option Verify="R" (or "RW").
  • The following attributes are checked during LVS: Resistance, Width, Strips for the option Verify="WL" (or "RWS").
