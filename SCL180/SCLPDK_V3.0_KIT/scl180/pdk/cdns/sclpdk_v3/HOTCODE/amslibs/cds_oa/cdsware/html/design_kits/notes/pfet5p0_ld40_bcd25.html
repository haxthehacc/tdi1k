DESCRIPTION :

'pfet5p0_ld40' implements the 20V-40V VDS pMOS transistor. The gate width can be modified and wide transistors can be generated as an array of parallel fets using the 'Number of Fingers' attributes. Multiple fets connected in parallel on the same active area are called 'fingers'.

ATTRIBUTES:

  • Fingers: Controls the number of gate fingers connected in parallel in the same active area.
  • Finger Width: Controls the width of a gate finger.
  • Length: Controls the effective channel active length.
  • Drift Length: Controls the effective channel active to drain active length (drift region).
  • Max Vds Estimate: Calculates the max Vds based off the ldrift value
  • RDS On Estimate: Calculates RDS On based off the width, ldrift, and number of fingers.
  • Thermal Resistance Option: A value of '1' activates the thermal resistance option in the simulation model.
  • Post Layout Parasitics?: A value of '0' enables parasitics and a value of '1' disables them in the simulation model. '1' will be the default value in the post-layout extracted view.

NOTES:

  • The total width of a transistor is given by: width = NF * WF
  • The total number of gates of a transistor is given by: gates = NF

LVS:

  • The following attributes are checked during LVS: width, length, drift region length, number of fingers.
