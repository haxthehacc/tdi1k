DESCRIPTION :

  • 'pCurMir' implements the current mirror with pfets. This device contains a custom layout pcell to optimize the layout of this structure, and several attributes control exactly how to draw the layout of the structure. For simulation purposes, this device contains a schematic view with 'pfet' devices that inherit properties from the pCurMir instance.

ATTRIBUTES:

  • Width (finger): The finger width to use for each pfet gate.
  • Length: The gate length to use for each pfet gate.
  • Number of Fingers: Number of fingers to use for each pfet gate.
  • Number of Slices: Number of slices to use for each pfet. Wide transistors can be split up into multiple sub-transistors ('slices') connected in parallel in the same pcell. 'Number of Slices' controls the number of parallel sub-transitors.
  • Current: Specifies the current through each pfet. The layout width of the source/drain metal will be adjusted to carry this current so as not to exceed maximum current density requirements.
  • Top Metal on Bulk: Controls which metal is used in the layout to connect the Bulk (substrate) nodes of the fets.
  • Top Metal on Src: Controls which top metal is used in the layout for the Source nodes of the fets.
