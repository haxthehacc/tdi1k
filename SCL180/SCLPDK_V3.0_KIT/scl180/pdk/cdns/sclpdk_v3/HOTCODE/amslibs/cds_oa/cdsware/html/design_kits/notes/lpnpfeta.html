DESCRIPTION :

  • 'lpnpfeta' implements a gate controlled lateral pnp transistor. The device has five terminals: collector, gate, emitter, base (or n-well) and substrate. The model is a 4-transistor equivalent sub-circuit model consisting of 3 bipolar pnp transistors: the intrinsic lateral pnp, two parasitic vertical pnps between the emitter/collector and the substrate and 1 pmos transistor between the emitter and collector of the device. During normal operation as a bipolar device, a positive voltage should be applied on the gate with respect to the emitter to prevent mos action.

ATTRIBUTES:

  • Substrate Node: The node to which the substrate of the device is connected. This attribute provides an implicit substrate pin for the device.
  • Count: Controls the number of instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic

NOTES:

LVS:

  • The following attributes are checked during LVS: a (area) and ne (Number of emitters).
