Here are the restrictions and use notes: - It uses Cadence layers metal1 - metal6, purpose rr - The marking layer must exactly cover the metal shape (coincide with two or more edges) to cause the desired break in connectivity. - Current implementation requires user to place an ideal resistor (not presistor) in schematic and manually calculate resistance desired for the schematic resistor. - There is no layout cell - the marking layer must be added manually. - Because of the restriction that this shape exactly match regular metal, it is likely to be error-prone to force a complex path to be covered with this marking layer unless a copy/paste is done. - Because of the implementation of the dummy resistor marking layer, it must EXACTLY match the size of the regular metal drawing shapes. The marking layer will be treated as regular metal drawing layer during mask prep. - The layout size of the resistor should be as small as possible since it affects the accuracy of the extraction tool. The dummy resistor area will be excluded from other parasitic calculations like lumped or distributed C. - The R value is not compared during LVS. </PLAINTEXT></BODY></HTML>
