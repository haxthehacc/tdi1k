DESCRIPTION :

'ind_oct' is an octagonal single-ended inductor. The Design Kit for inductors is closely integrated with a Matlab executable that is used to calculate performance and model parameters for any inductor given a set of geometric properties of the inductor.

ATTRIBUTES :

Taken from 'Edit Inductor Parameters' form

  • Inductor Type: Indicates the name of the Matlab model that this inductor is using - for the vast majority of inductors, this parameter cannot be changed.
  • Xsize: The total width of the inductor in the X direction.
  • Ysize: The total height of the inductor in the Y direction. Value is equal to Xsize.
  • Width: Controls the width of the inductor segments.
  • Spacing: Controls the space between inductor segments.
  • Num of Turns: Controls the total number of turns in the inductor. A quarter turn is .25, a full turn is 1. For instance, to create an inductor with four and a half turns, specify Num of Turns = 4.5.
  • Draw Shield: Implements ground shield
  • Polyimide: Adds polyimide
  • PackageEr: Sets the packaging dielectric constant
  • Underpass Geometry: Sets underpass geometry to 'straight' or 'perpendicular'.
  • Operating Freq: This specifies the application frequency at which the inductor is targeted for. This parameter is used by the Matlab executable from which to calculate some of the inductor performance parameters, such as Peak Q.
  • End Freq: This parameter specifies the high end of the frequency range for which to compute Y parameters by the Matlab executable.
  • Toggle Circuit Params Display: Toggling this button displays or hides all parameters associated with the model for this inductor as computed by the Matlab executable.

NOTES:

  • The inductor pcell generates a marking layer around the layout. This marking layer is used to identify the inductor during LVS.
  • No layer can be run over or under the inductor.

LVS:

  • The following attributes are checked during LVS:

    • Width, Spacing, Number of Turns, XSize and YSize
