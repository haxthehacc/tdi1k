These rules no longer apply. * Updated 7.S.2 to allow slot vias inside NPNs. ________________________ Date: 10/31/2012 Library: sbc18 Relevant Specification Documents: * Design Rules: NPB-PS-0179, Rev. 14 * Electrical Spec: NPB-PS-0267, Rev. 17 * Design Application Note for SBC18H4: NPB-PS-1106, Rev. 1 Process Synopsis: * The SBC18H4 process has 6 layers of aluminum metal and supports: - 1.8V and 3.3V MOSFETs, - high speed (340GHz) NPN with deep trench, - TiN resistor between metal3 and metal4, - poly resistors (silicided, low and high value unsilicided), - N-well resistor, - 2.8fF/um2 MIM capacitor between metal4/metal5 or metal3/metal4, - 5.6fF/um2 stacked MIM capacitor between metal3 and metal5, - square and octagonal inductors using 2.8um thick top metal6, - MOS varactor, - triple well and deep trench isolation. Software versions tested with this release: * Cadence IC release 5.10.41_USR6.127.29, 6.1.3.500.10 * Cadence Spectre release 10.1.1.092.isr7 * Synopsys Hspice release E-2010.12-ENG2 * Mentor Calibre release v2010.2_38.23 * Cadence Assura release 4.1_USR1_HF8 This is the initial release of the SBC18H4 kit. The SBC18H4 process differs from other SBC18 variants by using a grating and trim mask combination to form the NPN emitters. </PLAINTEXT></BODY></HTML>
