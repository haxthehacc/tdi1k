DESCRIPTION :

'ldiff_stk561' is a rectangular differential stacked solenoid fabricated in M5 & M6, Cu.

ATTRIBUTES :

  • Substrate Node : The node to which the substrate of the inductor is connected. This attribute provides an implicit substrate pin for the device.
  • Change input parameters : This button brings up the inductor calculation window. Electrical parameters for inductor models are automatically calculated from physical and functional parameters.
  • Inductor Type (Read Only) : Displays the type of inductor.
  • Number of Turns : Controls the total number of turns in the inductor. A quarter turn is .25, a full turn is 1. For instance, to create an inductor with four and a half turns, specify Num of Turns = 4.5.
  • Spacing : Controls the space between inductor segments.
  • Width : Controls the width of the inductor segments.
  • Xsize : The total width of the inductor in the X direction.
  • Ysize : The total height of the inductor in the Y direction.

NOTES:

  • The inductor pcell generates a marking layer around the layout. This marking layer is used to identify the inductor during LVS.
  • It is not allowed to run any layer over or under the inductor.

LVS:

  • For LVS to recognize the differential inductor, the marking layer must be intersected by two metal3 wires that connect to the P1 and P2 pins and one metal1 wire that connects to the TAP pin. It is not allowed to connect the pins in any other layer.
