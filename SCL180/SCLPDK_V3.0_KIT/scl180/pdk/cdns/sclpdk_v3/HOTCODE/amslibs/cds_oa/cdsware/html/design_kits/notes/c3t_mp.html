DESCRIPTION:

'c3t_mp' implements a capacitor fabricated using poly as the bottom plate and metal (tungsten-trinitride) as the top plate.

ATTRIBUTES:

  • Substrate Node: The node to which the substrate of the capacitor is connected. This attribute provides an implicit substrate pin for the device.
  • ToUse: Controls the way multiple parallel capacitors are specified. When 'ToUse' is set to 'multiplier', the Count attribute will be editable and the 'Columns in X' and 'Rows in Y' attributes will disappear. When 'ToUse' is set to 'col/row' the opposite will happen. See below for the meaning of these attributes.
  • Count: Indicates the number of capacitors connected in parallel. Specifying a 'Count' of n is equivalent to drawing n capacitor instances and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute is available when 'ToUse' is set to 'multiplier'. This attribute can only be used in the schematic.
  • Columns in X: When 'ToUse' is set to 'col/row' the capacitor will be implemented as an array of unit capacitors. 'Columns in X' specifies the number of columns of unit capacitors in the array.
  • Rows in Y: When 'ToUse' is set to 'col/row' the capacitor will be implemented as an array of unit capacitors. 'Rows in Y' specifies the number of rows of unit capacitors in the array.
  • Capacitor Calculation: This button brings up the capacitor calculation window
    • .
  • Calculate: Indicates the data entry mode of the capacitor calculator. A Capacitor is described by the attributes 'Capacitance', 'Unit Width', 'Unit Length' and 'Aspect Ratio'. Depending on the value of the 'Calculate' attribute, two of these attributes will be editable and the other two will be calculated : 'C' calculates 'Capacitance' and 'Aspect Ratio' based on 'Unit Width' and 'Unit Length'. 'L' calculates 'Unit Length' and 'Aspect Ratio' based on 'Capacitance' and 'Unit Width'. 'W' calculates 'Unit Width' and 'Aspect Ratio' based on 'Capacitance' and 'Unit Length'. 'WL' calculates 'Unit Width' and 'Unit Length' based on 'Capacitance' and 'Aspect Ratio'.
  • Unit Width (X): Controls the width of a unit capacitor in the capacitor array.
  • Unit Length (Y): Controls the length of a unit capacitor in the capacitor array.
  • Capacitance: Controls the total capacitance value of the capacitor array.
  • Unit ARatio = w/l: Controls the aspect ratio of a unit capacitor in the capacitor array.
  • Bottom Contacts: Controls the location of bottom plate contacts around each unit capacitor in the capacitor array. This location is specified with a string of characters indicating the sides where bottom plate contacts have to be added: 'l' means left, 'r' means right, 't' means top and 'b' means bottom. For instance, 'Bottom Contacts'=l will add a column of bottom plate contacts on the left side of the devices, 'Bottom Contacts'=lbrt will add a full ring of bottom plate contacts around the device. 'Bottom Contacts'=n generates a device without bottom plate contacts.
  • Bulk: The capacitor pcell support the use of a shield around the array. The shield The 'Bulk' attribute controls the location of the shield around the capacitor array. This location is specified with a string of characters indicating the sides where shields have to be added: 'l' means left, 'r' means right, 't' means top and 'b' means bottom. For instance, Bulk=l will add a shield on the left side of the array, Bulk=lbrt will add a full shield ring around the array. Bulk=n generates a device without shield.
