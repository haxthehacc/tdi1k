DESCRIPTION:

'nfet3p3_cascode' implements the nMOS dual gate (stacked) transistor configuration, which consists of two thick oxide 3.3V cascoded nfets.

ATTRIBUTES:

  • Multiplier: Controls the number of device instances connected in parallel. Specifying a 'Multiplier' of n is equivalent to drawing n instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Length1: Controls the length of the first fet.
  • Length2: Controls the length of the second fet.
  • Total Width: Controls the total width .
  • Finger Width: Controls the width of a gate finger.
  • Fingers: Controls the number of gate fingers connected in parallel in the same active area.
  • Threshold:When applied, maximum value allowed for 'Finger Width' in 'Total Width' calculation
  • Apply Threshold: Based on threshold value, calculates the number of fingers and finger width needed to equal the defined 'Total Width'
  • Gate Connection: Select the desired gate connection: None, Top, Bottom, Both, or Alternate.
    • None: No gate connection placed on any finger
    Top: Gate connection placed on the top of each finger
    Bottom: Gate connection placed on the bottom of each finger
    Both: Gate connection placed on the top and bottom of each finger
    Alternate: Gate connection alternates between top and bottom
  • Style: Specifies the layout style to be used for the device. The 'unconnected' layout style leaves the sources, drains, and gates of the transistors unconnected. The '1met' style uses metal1 to connect the source and drain areas. The '2met' and '2met(RF)' style use metal1 to connect the sources and metal2 to connect the drains.
  • Current: Specifies the current through the device. The width of the source/drain metal wires will be adjusted to carry this current.
  • S/D Metal Width: Controls the width of the source and drain. Used to calculate as, ad, ps, and pd.
  • Switch S/D: Ability to switch source and drain in layout.
  • Bodytie Type: Choose from: None, Integrated, or Detached. Integrated allows the user to have Left and/or Right Taps. Detached allows the user to have Left, Right, Top, Bottom Taps.
  • Edit Area and Perim: Makes the Source/Drain Area/Periphery fields editable.
  • Source/Drain Diffusion Area/Periphery: Area and periphery values for source and drain are always netlisted. These values are either internally computed or entered by the user.

NOTES:

LVS:

  • The following attributes are checked during LVS: wf (finger width), lf1 (Length1), lf2 (Length2) and nf (number of fingers).
