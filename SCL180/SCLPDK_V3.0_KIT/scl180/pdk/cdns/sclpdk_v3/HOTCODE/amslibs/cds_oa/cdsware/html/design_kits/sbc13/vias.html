DESCRIPTION:

  • The Contacts and Vias that end in "_ablb" have different enclosure and spacing rules. When a 3x3 array of contacts or vias is used, the spacing between the contacts and vias increases along with the enclosures. A 2x2 array follows the default spacing and enclosure of a single contact or via.

  • Multi Part Paths are supported in this Process. A path that consists of more than one part is called a multipart path.
       The MPP command can be found under the "Create" pulldown in Layout Window.
       MPP names end with "_mpp" and can be used to create PTAP and NTAP guard rings.
          If an incomplete ring is needed, be sure to edit MPP subparts and extend implant layer to minimum enclosure.
       MPPs can be used to create cont, v12, v23, v34 ,v45 and v56 line contacts as well as a ring.
       At this time, you must bring up the MPP form to set the contact or via you desire. (double click middle mouse button to bring up menu)
       All vias and contacts are choppable.
