DESCRIPTION :

'pfet_pass' implements the high density, massively parallel pMOS pass gate transistor.
The cell is created as a configurable array of parallel fet base cells. Each base cell consists of a specified number of fingers on a specified number of Active slices.

USAGE NOTES:

Number of Fingers must be even.
Total Gate Width = (Base Cell Rows X Base Cell Columns X Number of Slices X Number of Fingers X Finger Width).
We suggest using about 2 for the "Number of Slices" parameter because the generator must draw each sliced individually. Specifying more than that drastically increases generator execution time, but will still work.

ATTRIBUTES:

  • Count: Controls the number of fet instances connected in parallel. Specifying a 'Count' of X is equivalent to drawing X pfet_pass instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be generated as X separate layout cells during a "gen from source" command. This attribute is only meaningful in the schematic.
  • Rows of Base Cell: Controls the number of rows in the Array of the Base Cell.
  • Columns of Base Cell: Controls the number of columns in the Array of the Base Cell.
  • Width (finger): Controls the width of a single gate finger.
  • Length: Controls the gate length.
  • Number of Fingers: Controls the number of gate fingers connected in parallel in the same active area ( a slice ).
  • Number of Slices: Number of Active islands with shorted gates in the Base Cell.
  • Draw Via Placeholder: If selected, the generator will draw via place holders on the layout view for use with the Diva code to Generate any combination of Source Vias, Drain Vias, and Metal 2 you specify through the Diva switches.

NOTES:

LVS:

  • The following attributes are checked during LVS: width, length.
