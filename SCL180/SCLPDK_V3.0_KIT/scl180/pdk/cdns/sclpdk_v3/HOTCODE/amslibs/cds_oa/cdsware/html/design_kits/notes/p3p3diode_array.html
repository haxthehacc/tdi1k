DESCRIPTION :

'p3p3diode_array' implements an array of ptype junction diodes.

ATTRIBUTES:

  • Count: Controls the number of arrayed instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n arrayed instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Style: Controls whether the interconnect scheme in the layout pcell is performed using metal 1 or metal 2 layer.
  • Columns in X: Controls number of columns of diodes to use in the x direction in the layout pcell. Note that this affects the Total Area and Perimeter of the structure.
  • Rows in Y: Controls number of rows of diodes to use in the y direction in the layout pcell. Note that this affects the Total Area and Perimeter of the structure.
  • Unit Width: Controls the width of an individual p3p3diode structure in the array. Note that this affects the Total Area and Perimeter of the structure.
  • Unit Length: Controls the length of an individual p3p3diode structure in the array. Note that this affects the Total Area and Perimeter of the structure.

NOTES:

LVS:

  • The following attributes are checked during LVS : area, perimeter.
