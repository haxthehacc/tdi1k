DESCRIPTION:

'var_ni' implements a varactor from the EB junction of a vertical PNP transistor without an extra implant. Capacitance and Conductance both scale with the number of fingers (neglecting external series resistance), and so the Varactor Q is independent of capacitance.

ATTRIBUTES:

  • Count: Controls the number of var_ni instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n var_ni instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Width of anode: Controls the width of the anode fingers (p+ diffusion).
  • Length of anode: Controls the length of the anode fingers (p+ diffusion).
  • Number of anodes: Controls the number of anode fingers connected in parallel. The number of cathodes will be equal to the number of anodes + 1.

NOTES:

LVS:

  • The following attributes are checked during LVS: Width of anode, Length of anode, Number of anodes.
