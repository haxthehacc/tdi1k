DESCRIPTION :

'l3t' is a rectangular spiral inductor fabricated in top metal. The Design Kit for inductors is closely integrated with a Matlab executable that is used to calculate performance and model parameters for any inductor given a set of geometric properties of the inductor.

ATTRIBUTES :

  • Skin Effect Model: Available skin effect models for the inductor subckt. Possible values are: Parallel RL and Resistor Ladder (selected  libraries only).
  • Substrate Node: The node to which the substrate of the inductor is connected. This attribute provides an implicit substrate pin for the device.
  • Change input parameters: This button brings up the inductor calculation window whose parameters are outlined below. User should enter all input parameters via this form. Once OK or Apply is selected, a Matlab executable is called to calculate the inductor's electrical characteristics based on the user's geometric inputs. These calculated inputs are then returned back to Cadence environment, and are used by the simulation model of the inductor to accurately simul ate characteristics of the inductor.
  • Toggle Circuit Params Display: Toggling this button displays or hides all parameters associated with the model for this inductor as computed by the Matlab executable.
  • Other: There are a whole set of other parameters that are displayed but not editable. These are for information purposes only. Consult the above referenced link for details on these parameters.

    • "Edit Inductor Parameters" Form Parameters:

    • Inductor Type: Indicates the name of the Matlab model that this inductor is using - for the vast majority of inductors, this parameter cannot be changed.
    • Search Inductors: When enabled, this button provides new window to search  for inductors by electrical parameters like Q, L, or frequency.
    • Num of Turns: Controls the total number of turns in the inductor. A quarter turn is .25, a full turn is 1. For instance, to create an inductor with four and a half turns, specify Num of Turns = 4.5.
    • Spacing: Controls the space between inductor segments.
    • Width: Controls the width of the inductor segments.
    • Xsize: The total width of the inductor in the X direction.
    • Ysize: The total height of the inductor in the Y direction.
    • Operating Freq: This specifies the application frequency at which the inductor is targeted for. This parameter is used by the Matlab executable from which to calculate some of the inductor performance parameters, such as Peak Q.
    • Start Freq: (JIT 1 only) The Matlab executable computes a set of Y parameters from which to derive all calculations including inductor performance parameters. This parameter specifies the low end of the frequency range for which to compute Y parameters for by the Matlab executable. It is automatically calculated for JIT 2 and later.
    • End Freq: This parameter specifies the high end of the frequency range for which to compute Y parameters for by the Matlab executable.
    • Freq Step: (JIT 1 only) This parameter specifies the resolution of the frequency range for which to compute Y parameters for by the Matlab executable. The smaller this number, the higher the resolution, and the better the accuracy of the Matlab computations. Increased resolution will come at the expense of computation times. It is automatically calculated for JIT 2 and later.
    • Draw Shield: Enable or disable generation of shield pattern under inductor.
    • Polyimide: Enable or disable calculation of polyimide overcoat effects on inductor performance.
    • Package Dielectric: Specify package dielectric (Er) value from 1.0 to 10.0.

NOTES:

  • 'spiralinduct' and 'l3t' are the same component.
  • The inductor pcell generates a marking layer around the layout. This marking layer is used to identify the inductor during LVS.
  • It is not allowed to run any layer over or under the inductor.

LVS:

  • For LVS to recognize the inductor, the marking layer must be intersected by one top-metal wire that connects to the P1 pin and one top-metal or top-metal - 1  wire that connects to P2 pin. It is not allowed to connect the pins in any other layer.
  • The following attributes are checked during LVS:

    • Width, Spacing, Number of Turns, XSize and YSize
