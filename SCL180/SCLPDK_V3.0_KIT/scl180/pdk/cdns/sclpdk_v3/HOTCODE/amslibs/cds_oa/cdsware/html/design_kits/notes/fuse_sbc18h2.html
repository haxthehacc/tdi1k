DESCRIPTION:

'fuse' implements the fuse resistor device.

ATTRIBUTES:

  • Fuse Mode: Controls whether the fuse is set to Open, Closed, or Other.

    • Open: Value = 1M
    Closed: Value=0.4
    Other: Value is defined by the user

  • Value: Value of fuse.
