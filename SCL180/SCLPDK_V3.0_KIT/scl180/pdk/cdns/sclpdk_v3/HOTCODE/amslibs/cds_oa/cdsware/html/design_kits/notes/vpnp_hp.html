DESCRIPTION:

'vpnp_hp' implements a high Ft vertical pnp bipolar transistor. The symbol allows the user to select from 2 widths and vary the length, but currently only fixed layouts will be available. The following fixed layouts can be placed: layout_0p8x5 (0.8ux5u), layout_0p8x10 (0.8ux10u), layout_0p8x20 (0.8ux20u), layout_2p5x5 (2.5ux5u)

ATTRIBUTES:

  • Model Type Currently only 'gp' (gummel poon) is available
  • Emitter Width/Model Choose from the following choices: 0.8u/vpnpn, 2.5u/vpnpw
  • Emitter Length Controls the emitter length
  • Multiplier: Controls the number of transistor instances connected in parallel. Specifying a 'Multiplier' of n is equivalent to drawing n transistor instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Temprise Temperature rise from ambient
  • Layout nameView name of fixed layout which will be used. Limited to layouts mentioned in cell description
