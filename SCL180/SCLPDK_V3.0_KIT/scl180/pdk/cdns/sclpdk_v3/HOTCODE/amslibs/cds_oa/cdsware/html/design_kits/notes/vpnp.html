DESCRIPTION:

'vpnp' implements a vertical pnp bipolar transistor. The layout is fixed with multiple emitter dimensions.

ATTRIBUTES:

  • Emitter Size: User selects from the following choices: 3ux3u, 5.4ux5.4u, 11ux11u, 25ux25u
  • Multiplier: Controls the number of transistor instances connected in parallel. Specifying a 'Multiplier' of n is equivalent to drawing n transistor instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
