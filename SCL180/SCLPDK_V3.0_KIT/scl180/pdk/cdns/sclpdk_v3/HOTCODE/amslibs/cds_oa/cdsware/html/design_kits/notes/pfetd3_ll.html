DESCRIPTION :

'pfetd3_ll' implements the thin oxide low-leakage pMOS transistor. The gate width and length can be modified.

ATTRIBUTES:

  • Substrate Node: The node to which the substrate of the fet is connected. This attribute provides an implicit well pin for the device.
  • Count: Controls the number of fet instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n fet instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • Style: Specifies the layout style to be used for the device. The 'unconnected' layout style leaves the sources, drains and gates of the transistors unconnected. The '1met' style uses metal1 to connect the source and drain areas. The '2met' is the same as the 1met style for fets with 1 slice (as are all fets entered with the pfetd parameter sheet.)
  • Width (finger): Controls the width of a gate finger.
  • Length: Controls the gate length.

NOTES:

LVS:

  • The following attributes are checked during LVS: width, length.
