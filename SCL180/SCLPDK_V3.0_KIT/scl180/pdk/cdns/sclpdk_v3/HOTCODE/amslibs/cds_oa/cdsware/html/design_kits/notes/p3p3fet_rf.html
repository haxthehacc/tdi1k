DESCRIPTION :

'p3p3fet_rf' implements the thick oxide 3.3V version of the RF pMOS transistor. The gate width and length can be modified, and wide transistors can be generated as an array of parallel fets using the 'Number of Fingers' and 'Number of Slices' attributes. Multiple fets connected in parallel on the same active area are called 'fingers'. Very wide transistors can be split up in multiple multi-finger sub-transistors that are connected in parallel. The resulting sub-transistors are called 'slices'.

ATTRIBUTES:

  • Substrate Node: The node to which the substrate of the fet is connected. This attribute provides an implicit substrate pin for the device.
  • Count: Controls the number of fet instances connected in parallel. Specifying a 'Count' of n is equivalent to drawing n fet instances in the schematic and connecting them in parallel. In VirtuosoXL the device will be implemented as n separate layout cells. This attribute can only be used in the schematic
  • On Grid: Selecting this button puts the device pins on a .5um grid.
  • Width (finger): Controls the width of a gate finger.
  • Length: Controls the gate length.
  • Number of Fingers: Controls the number of gate fingers connected in parallel in the same active area.
  • Number of Slices: Wide transistors can be split up into multiple sub-transistors ('slices') connected in parallel in the same pcell. 'Number of Slices' controls the number of parallel sub-transitors.
  • Current: Specifies the current through the device. The width of the source/drain metal wires will be adjusted to carry this current.
  • Bulk: Indicates the location of substrate ties. This location is specified with a string of characters indicating the sides where substrate ties have to be added: 'l' means left, 'r' means right, 't' means top and 'b' means bottom. For instance, Bulk=l will add a substrate tie on the left side of the device, Bulk=lbrt will add a full substrate tie ring around the device. Bulk=n generates a device without substrate ties.

NOTES:

  • The total width of a transistor is given by: width = COUNT * NF * SLICES * WF
  • The total number of gates of a transistor is given by: gates = COUNT * NF * SLICES

LVS:

  • The following attributes are checked during LVS: finger width, finger length, number of fingers.
